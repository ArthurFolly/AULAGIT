# AULAGIT
## Iniciando Projeto GIT
- Repositório Local
- Repositório Remoto
- git config -- global user.name "(nome)"
- git config -- global user.mail "(email)"
- git init
- git remote add origin "link do repositorio"

Faz umas coisas ai

-git push  origin main | origin master
## Clone, Fetch e Pull
-git clone (repositorio)
-git fetch
-git status
